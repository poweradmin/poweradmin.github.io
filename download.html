Getting Poweradmin

Normal release

The most recent version is Poweradmin 2.1.7, which was released on Jun 10th, 2014. It can be downloaded at:

https://sourceforge.net/projects/poweradmin/files/latest/download?source=directory

The announcement can be found here.

Latest code

If you have an adventurous life style and you want to run a bleeding edge version of Poweradmin, you may want to download the latest code. GitHub creates automatically a tarball of the latest revision of the regular Poweradmin code. These should be fairly stable, but nevertheless you shouldn't use these for production systems. The latest version can be downloaded from:

https://github.com/poweradmin/poweradmin/zipball/master

Access to source code

You can download the code from the links mentioned above. To checkout the latest version use Git:

git clone git://github.com/poweradmin/poweradmin.git
or
git clone https://github.com/poweradmin/poweradmin.git
