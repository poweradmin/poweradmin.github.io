Getting Poweradmin

Normal release

The most recent version is Poweradmin 2.1.6, which was released on May 7th, 2012. It can be downloaded at:

https://github.com/downloads/poweradmin/poweradmin/poweradmin-2.1.6.tgz

The announcement can be found here.

Latest code

If you have an adventurous life style and you want to run a bleeding edge version of Poweradmin, you may want to download the latest code. GitHub? creates automatically a tarball of the latest revision of the regular Poweradmin code. These should be fairly stable, but nevertheless you shouldn't use these for production systems. The latests version can be downloaded from:

https://github.com/poweradmin/poweradmin/zipball/master

Access to source code

You can download the code from the links mentioned above. To checkout the latest version use Git:

git clone git://github.com/poweradmin/poweradmin.git
or
git clone https://github.com/poweradmin/poweradmin.git
